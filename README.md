# simulador-cache
Trabalho para a disciplina de Arquitetura Organizacional de Computadores II na UFPel, ministrada pelo professor Julio Carlos Balzano de Mattos.


O trabalho consiste em implementar um simulador de cache, dando ao usuário a possibilidade de escolher o número de sets, tamanho do bloco e associatividade, além de enviar um arquivo em binário com endereços de 32 bits. No final da execução, é mostrado um relatório com quantidade de hits e misses, além da hit ratio e miss ratio. 


O input do programa se dá pela linha de comando ```python cache_simulator.py <nsets_L1>:<bsize_L1>:<assoc_L1> enderecos.bin```.
